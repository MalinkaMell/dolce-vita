# Dolce Vita

###  Italian Gelato and Grocery 

Design/development for local Italian gelato and grocery store. Work in progress!

### Technologies/Npm packages used:

* JavaScript/ES6 / HTML5 / CSS3
* NodeJS
* ReactJS
* React Router
* Bootstrap / React Bootstrap / Mdbreact 
* Express
* Axios 
* Dotenv
* Nodemailer
* Yelp fusion
